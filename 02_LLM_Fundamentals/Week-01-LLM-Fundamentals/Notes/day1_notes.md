## 📚 What I Understood

### 1. Large Language Model (LLM)
- An LLM is an AI model trained on a huge amount of text data.
- It can understand, generate, summarize, translate, and answer questions in natural language.
- Examples include GPT, Llama, and Gemini.

### 2. API Basics
- An API allows two software applications to communicate.
- My application sends a request to the OpenAI API.
- The API forwards the request to the AI model and returns the generated response.

### 3. Input and Output
- **Input** is the prompt or instruction sent to the AI.
- **Output** is the response generated by the AI model.

### 4. Token
- A token is the smallest unit of text processed by an LLM.
- A token can be a word, part of a word, punctuation, or special character.

### 5. Tokenization
- Tokenization is the process of converting text into tokens before the AI processes it.

### 6. Context Window
- A context window is the maximum number of tokens an AI model can consider at one time.
- If the conversation becomes too long, older tokens may be removed from the active context.

### 7. Temperature
- Temperature controls how creative or random the AI's responses are.
- Low temperature (e.g., 0.2) gives more focused and consistent answers.
- High temperature (e.g., 1.0) gives more creative and varied answers.

## 💡 What I Understood

- LLMs generate text by predicting one token at a time.
- APIs make it easy to use powerful AI models without building them from scratch.
- Every prompt is converted into tokens before being processed.
- The context window limits how much information the model can remember during a conversation.
- Temperature changes the style of the AI's responses but not its knowledge.

## 📝 Mini Challenge Answer

**Why can't an LLM remember every conversation forever?**

An LLM cannot remember every conversation forever because storing every user's conversations would require a huge amount of memory and computing resources. It would also raise privacy and security concerns. Instead, the model only processes the information available within its context window. When the conversation becomes too long, older information may no longer be available unless it is summarized or stored separately by the application.

## 🎤 Interview Questions

### Q1. What is a Large Language Model (LLM)?

An LLM is an AI model trained on massive amounts of text data to understand and generate human-like language.

### Q2. What is the difference between Machine Learning and an LLM?

Machine Learning is a broad field where computers learn from data. An LLM is a specialized Machine Learning model designed specifically for understanding and generating natural language.

### Q3. What is a Token?

A token is the smallest unit of text that an AI model processes. It may be a word, part of a word, punctuation, or another text fragment.

### Q4. What is Tokenization?

Tokenization is the process of breaking text into tokens before the AI processes it.

### Q5. What is a Context Window?

A context window is the maximum number of tokens an AI model can consider while generating a response.
