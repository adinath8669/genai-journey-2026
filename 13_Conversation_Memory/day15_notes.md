# 💡 What I Understood

## 1. What is Memory?

Memory is the ability of an AI application to remember previous messages in a conversation. The LLM itself does not permanently remember anything. Instead, our application stores the conversation and sends it back to the model with every new request.

---

## 2. What is Conversation History?

Conversation history is the collection of all previous messages exchanged between the user and the AI.

Example:

```text
User: Hi

AI: Hello!

User: My name is Adinath.

AI: Nice to meet you.

User: What is my name?
```

Instead of sending only the last question, the application sends the entire conversation history so the model can answer correctly.

---

## 3. What is HumanMessage?

HumanMessage represents the message sent by the user.

Example:

```python
HumanMessage(content="What is AI?")
```

Every question or instruction from the user is stored as a HumanMessage.

---

## 4. What is AIMessage?

AIMessage represents the response generated by the AI model.

Example:

```python
AIMessage(content="AI stands for Artificial Intelligence.")
```

Every response from the model is stored as an AIMessage.

---

## 5. What is SystemMessage?

A SystemMessage defines how the AI should behave throughout the conversation.

Example:

```python
SystemMessage(
content="You are a helpful AI assistant.")
```

It sets the personality, tone, and rules for the model before the conversation begins.

---

## 6. Message History

Message history is simply a list of all messages in order.

Example:

```text
SystemMessage
↓

HumanMessage

↓

AIMessage

↓

HumanMessage

↓

AIMessage
```

This history is sent to the LLM every time a new question is asked.

---

## 7. Stateless vs Stateful LLMs

### Stateless

A stateless application does not remember previous conversations.

Example:

User:
My favorite language is Python.

Later...

User:
What is my favorite language?

AI:
I don't know.

---

### Stateful

A stateful application stores previous messages and sends them again with every request.

Example:

User:
My favorite language is Python.

Later...

User:
What is my favorite language?

AI:
Your favorite language is Python.
